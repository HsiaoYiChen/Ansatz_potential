# Ansatz_potential
This code is developed to compute the exchange-correlation potential in Kohn-Sham equation of any arbitrary spin-orientation.
The detail is described in the Section IIA of the paper: https://journals.aps.org/prx/abstract/10.1103/PhysRevX.15.011054

New publication computing the magnon based on Ansatz Potential method: https://arxiv.org/abs/2510.14405
